The Frontend Basics
A robust app building engine

Setup

In order to setup the environment, developers should have the following dependencies pre-installed.

  • Ruby 1.9.3++ ( on Mac requires XCode or XCode CLT )
  • NodeJS 0.8.0++ and NPM
  • Text Editor
  • Chrome Web Browser

Developers should also have accounts with the following services

  • Amazon Web Services
  • Github

Cloning

If Appbot has already created the enterprise platform, then all you need to do is clone the Frontend Server Repository into your development box ( aka: computer )

$ git clone [repositoryURL]

This will "download" the code into your computer on the folder [respositoryName]

Once the repository is clone, and inside the server folder run:

$ npm install . //make require sudo

This will install all necessary dependencies using NPM

Testing

Testing, according to TDD is the first thing that must be done. So let's test out code first.

$ grunt test

This command will run all necessary test

Now we want to always have a terminal tab open with automated testing when any files change.

$ grunt watch test

Running

In order to run the application on the local environment, the first step we must take is to add the environment variables so that we can connect to the Api Server and such. Edit the configuration file and type all required Environment Variables

$ mate .configuration

Ask your team coordinator if you don't know the environment variables.

Start the Server

$ grunt server

We are starting a simple web server and a watch process to automatically compile ....

Open localhost:9900 in order to view the index page of the Application.

Deploying

Before deploying we must make sure all of our test pass and that it's got above 80% coverage.

In order to deploy we must use git to send the changes to the origin repository.

$ git push origin master

Note the proper use of Git is required in terms of commits and branches.

Once the origin repository gets the code it will merge it with the actual state of the repository and the run all test.
